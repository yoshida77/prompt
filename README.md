# Prompt

プロンプト研究


## Learnings

- Tokenにノイズを少なくすることが重要である
- あるべきものとは、人間の関係において、何が不足しているかを満たす構造
- 自分が欲しいものを根源的に考えていき、深ければ深いほど、全員に刺さる

### Reasoning / Non-Reasoning

- 非推論モデルを駆使して回答を設計し、推論モデルを使う
- 推論モデルを使って調査をして非推論モデルのための準備に使う
